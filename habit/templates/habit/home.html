Добро пожаловать в проект по управлению привычками!

Эффективное управление привычками помогает достигать целей и улучшать качество жизни.

Описание задач

Этот проект включает в себя создание и управление привычками с учетом различных аспектов, таких как:

  • Модели привычек: определение действия, времени, места и других параметров.
  • Эндпоинты для работы с фронтендом, включая регистрацию, авторизацию и управление привычками.
  • Интеграция с Telegram для рассылки уведомлений о привычках.
  • Пагинация и права доступа для удобного управления и просмотра данных.
  • Документация API для фронтенд-разработчиков.
